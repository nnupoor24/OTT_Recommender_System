PROJECT TITLE
A Personalized Content Recommendation System Using Behaviour Analytics for OTT Platforms


PROJECT DESCRIPTION
This project is a Netflix-like movie recommendation system that suggests personalized movies to users based on movie genres and user behaviour analytics.

The system uses machine learning techniques to analyze movie similarity and recommend relevant content.


TECHNOLOGIES USED
- Python
- VS Code
- Pandas
- NumPy
- Scikit-learn


DATASET USED
MovieLens Dataset


FEATURES
- Movie recommendation system
- Content-based filtering
- Genre similarity analysis
- User behaviour analytics
- Personalized OTT recommendations


ALGORITHM USED
- TF-IDF Vectorization
- Cosine Similarity


HOW TO RUN THE PROJECT
1. Open project in VS Code
2. Open terminal
3. Run command:

python src/main.py


EXPECTED OUTPUT
The system recommends movies similar to user preferences.
